# Marucci Elite
Sign up form

## Requirements
- MongoDB
- Node.js
- NPM

## Install

Install mongodb via homebrew
```
$ ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"
$ brew update
$ brew install mongodb
```

Get project files
```
$ git clone https://github.com/Marucci-Elite/signup.git
```

## Run
```
$ cd signup
$ node server.js
```
